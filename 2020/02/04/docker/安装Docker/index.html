安装Docker

引用自《Docker系列教程02-Docker安装》- 周立

Ubuntu

系统要求

  • Docker支持以下版本的Ubuntu，要求64位。

    • Bionic Beaver 18.04 (LTS)
    • Zesty 17.04
    • Xenial 16.04 (LTS)
    • Trusty 14.04 (LTS)

  • 支持运行的平台：x86_64、armhf、s390x(IBM Z)。其中，如选择IBM Z，那么只支持Ubuntu Xenial以及Zesty。

  • 本文使用Ubuntu 16.04 LTS，下载地址：http://cn.ubuntu.com/download/
